# 🌱 Smart Farm Advisor

An AI-powered crop recommendation system that helps farmers choose the best crop based on soil and weather conditions.

## 🚀 Features
- 🌤️ Real-time weather data (Temperature & Humidity)
- 🎤 Voice input for location (optional)
- ⚡ Auto-fill soil data for quick testing
- 🌾 AI-based crop prediction using Machine Learning

## 🧠 Technologies Used
- Python
- Streamlit
- Machine Learning (Scikit-learn)
- OpenWeatherMap API
- Speech Recognition

## 📊 Inputs
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

## 🎯 Output
- Recommended crop for farming

## ▶️ How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py

## 🌍 Use Case
This project helps farmers make better decisions and improve crop yield using AI.
